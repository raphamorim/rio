Rio

See it on Github

Why Rio?

Rio is built over WGPU and Rust. So, it's based on the WebGPU API, which makes it suitable for general purpose graphics and compute on the GPU.

Applications using wgpu run natively on Vulkan, Metal, DirectX 11/12, and OpenGL ES; and browsers via WebAssembly on WebGPU and WebGL2.

Is it ready?

Not yet, but you can submit your email to know when it is ready.
